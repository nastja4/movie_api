myFlix

API Usage Guide

Introduction:

The myFlix is a web application that provides users with access to information about different movies, directors, and genres.
The API provides an opportunity for Users to sign up, update their personal information, and create a list of their favorite movies.


Business Logic URL HTTP Method Query parameters Request body data fomat Response body data format
Return a list of ALL movies to the user /movies GET None None A JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user /movies/[Movie_Title] GET :Title None A JSON object holding data about a single movie
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/[Genre]/[Genre_Name] GET /genre/:Name None A JSON object with a description of the movie genre
Return data about a director (bio, birth year, death year) by name /movies/[Director]/[Director_Name] GET /director/:Name None A JSON object holding data about the director. Example:
{
Name: "",
Bio: "",
Birth: "",
Death: ""
}
Allow new users to register /users POST None A JSON object holding data about the user. Example:
{
"name": "",
"favoriteMovies": [""]
} 		A JSON object holding data about the registered user. Example:

Allow users to update their user info (username) /users/[ID] PUT :id A JSON object holding data about the new Username. Example:
{
"name": "Updated_Name",
, "favoriteMovies": [""]
} 		A JSON object holding data about the Username. Example:
{
"id": "",

Allow users to add a movie to their list of favorites /users/[ID]/[Movie_Title] POST /:id/:movieTitle None A text message informing that a movie has been added to the user's favorite movies
Allow users to remove a movie from their list of favorites /users/[ID]/[Movie_Title] DELETE /:id/:movieTitle None A text message informing that a movie has been removed from the user's favorite movies
Allow existing users to deregister /users/[ID] DELETE :id None A text message that a user email has been removed from the database
